# Morning Brief (ElevenLabs Worldwide Hackathon)

Morning Brief delivers a personalized daily podcast update tailored to your specific interests, leveraging AI web search to curate the newest happenings since your last listen.

This project integrates tools like ElevenLabs for high-quality text-to-speech synthesis, alongside services like claude and exa.ai for content processing.

The goal is to provide users with a custom-made audio briefing every morning on their favorite topics. You can check out the live demo at <https://morning-brief.up.railway.app/> and see a demonstration of the project in action via the video link provided.
